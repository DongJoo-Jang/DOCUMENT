REDIS : Remote Dictionary Server
원격 인메모리 DB

WAS : Web Apllication Server
웹 어플리케이션과 서버 환경을 만들어 동작시키는 기능을 하는 프레임워크
(웹 어플리케이셔을 실행시켜 기능을 수행하고 그 결과를 웹 서버에게 전달)


MELSEC : 미쯔비시 PLC 연결모듈
MODBUS : PLC 연결 모듈 (거의 표준) 

CDN : Client Delivery Network
정적 데이터 전송 서버 ( 라이브러리 다운 필요없이 가져다 쓸 때 쓴다)

3계층형 시스템 (웹 서버 AP서버 DB 서버)
클라이언트PC ➜ 웹 서버 ➜ AP 서버 ➜ DB 서버 ➜ AP서버 ➜ 웹 서버 ➜ 클라이언트PC

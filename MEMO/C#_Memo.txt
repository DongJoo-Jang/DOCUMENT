.aspx파일에서 <%= Session["key"] 먹는다

GetType 은  박싱되기 전 값으로 나온다
ex) int a = 10;
    object b = a ; b.GetType() == int 

IEnumerable 구현 객체  foreach 가능

null 집합 연산자 ??   널이 아니면 앞에꺼 널이면 뒤에꺼
ex)    string str1 = null;
            string str2 = str1 ?? "없음";

            Console.WriteLine($"str1 :{str1} , str2 : {str2}");
